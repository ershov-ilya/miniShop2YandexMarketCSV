miniShop2 to YandexMarket .CSV Export
===================
MODX Revo, miniShop2
--------------------

usage in Resource body (content type "text") call the snippet

```
[[!miniShop2YandexMarketCSV?
&parents=`58`
&productTemplateID=`7`
]]
```

where &parents - root of Catalog

productTemplateID - ID of product template
